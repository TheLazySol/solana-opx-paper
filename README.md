## Getting Started

### Prerequisites

- Node v20.13.1 or higher
- Rust v1.84.0 or higher
- Anchor CLI 0.30.1 or higher
- Solana CLI 2.0.23 or higher

### Installation

#### Clone the repo

```shell
git clone <repo-url>
cd <repo-name>
```

#### Install Dependencies

```shell
pnpm install
```

#### Commands

Start the web app

```shell
pnpm dev
```

Build the web app

```shell
pnpm build
```
